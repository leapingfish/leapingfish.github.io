Review: jQuery Plugins

  • Rules of the game:

    • It's important to pay attention to the order of how things are loaded. jQuery is the biggest dependency so it must come first. Then the plugin library. Then your code that runs the plugin.

  • Plugins to check out:

    • Slider: http://kenwheeler.github.io/slick/
    • Fullpage JS: https://github.com/alvarotrigo/fullPage.js#fullpagejs

Lab: Weather App

  • Create a Temp Converter will take celsius temperature, convert it to Fahrenheit and change the background image according to the weather.

  • Example Weather App
  • As a class, let's write our feature requirements necessary to create a fully functional application that takes celsius temperature, converts it to Fahrenheit and changes the background to match said temperature.
  • In groups of four write Pseudo-code for the application.
  • In pairs write the code to convert Celsius into Fahrenheit, and display the result in the browser.
  • Once results are being displayed in the browser, add the background changing functionality.
