    Javascript Data Types

    • Primary (primitive) Data Types - Primitive data types are all data types that are not objects
    • Boolean = true or false
    • Undefined = declared but never defined
    • Null = no value (is an assigned value)
    • Number = any number
    • String = a collection of characters, usually wrapped in parenthesis

    The Most Important Piece of Javascript (Arguably)

    • Object - An object is a collection of properties, and a property is an association between a name (or key) and a value. A property's value can be a function, in which case the property is known as a method. In addition to objects that are predefined in the browser, you can define your own objects. Pay specific attention to the syntax here - it is the important part!
    • var car = {
      wheels: 4,
      driver: 1,
      engine: 1,
      doors: 5,
      radio: 1,
      name: 'The Love Bug'
      }
    • How to get information out of an Object?
    • Dot notation: car.name = 'The Love Bug'
    • Bracket notation: car['name'] = 'The Love Bug'

Introduction to jQuery

  • jQuery is a JavaScript framework, which purpose is to make it much easier to use JavaScript on your website. You could also describe jQuery as an abstraction layer, since it takes a lot of the functionality that you would have to write many lines of JavaScript to accomplish and wraps it into functions that you can call with a single line of code. It's important to note that jQuery does not replace JavaScript, and while it does offer some syntactical shortcuts, the code you write when you use jQuery is still JavaScript code.

  • jQuery Syntax: $(tag/selector).method(arguments);

  • (jQuery JS) var parent = $('#about').parent();

  • (Vanilla JS) var parent = document.body.getElementById('about').parentNode;
