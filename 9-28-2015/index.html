New Data Structure: Arrays

  • Arrays are simplified list like objects.
  • How to create an Array:
    • var data = new Array('item1', 'item2', 'item3'); // this is rarely used
    • var data = ['item1', 'item2', 'item3'] // preferred method
  • How to retrieve item from an Array:
    • console.log(data[0]); // 'item 1'
  • PRACTICE
    • Save a list of the months in an array.
