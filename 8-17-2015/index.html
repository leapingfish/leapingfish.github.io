What is HTML5:

  • HyperText Markup Language
    • A language for displaying content (text, images, table, links, etc..)
    • Provides structure for your site
    • Interpreted by browser based on DOCTYPE
  • HTML5 was standardized 2014
  • HTML4 was standardized 1997
  • HTML was invented by Tim Berners-Lee in 1989
    • Tim Berners-Lee founded and directs World Wide Web Consortium (W3C)
    • The W3C oversees the Web's continued development

Give me an example teach! View example: HTML5 Markup


HTML ELEMENTS
  • HTML is made up of multiple elements (a.k.a. tags)
  • Most elements have an opening and closing tag.
    • <b>make it bold</b>
    • make it bold
  • But some elements do not have a closing tag
    • <img src="http://leapingfish.io/HeyArnold.jpg" alt="Hey Arnold">
  • Basic elements include:
    • <title></title>
    • <h1></h1> - heading 1
    • <p></p> - paragraph
    • <a></a> - link
    • <ul> <li>first list item</li> <li>second list item</li> </ul> - unordered list
    • <div></div> - basic container element
  • Check out the MDN documentation of elements for a full list

HTML ATTRIBUTES
  • Different elements have attributes that can define characteristics of that element
  • <tag attribute="value">Content</tag>
  • Common attributes include:
    • href
    • style
    • class
    • id
    • src
  • Quick examples:
    • <img src="photo.jpg">
    • <a href="http://www.google.com">Google</a>
    • <p class="my-class">Awesome Class</p>
    • <h1 style="color: red;">Intense!</h1>

HTML Named character references

Named character references (often casually called entities) are used to print characters that have a special meaning in HTML.

  • &lt; denotes the less-than sign (<)
  • &amp; denotes the ampersand (&)
  • &ldquo; denotes open quote (“)
  • &hearts; denotes a heart symbol (♥)

    Specificity! It will one of the most important things you'll need understand about CSS and therefore, will be your nemesis! Specificity determines which CSS rule is applied by the browsers.

    There are four distinct categories which define the specificity level of a given selector: inline styles, IDs, classes, and elements. Example Please my good sir: Say you have an unordered list of drinks and you want to put a special style on your favorite.
