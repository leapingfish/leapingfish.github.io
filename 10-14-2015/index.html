What is a Form?

A webform, web form or HTML form on a web page allows a user to enter data that is sent to a server for processing. Forms can resemble paper or database forms because web users fill out the forms using checkboxes, radio buttons, or text fields.

  • Important Form Tags

  • <form></form>

    • attributes

    • action="file_to_process_our_form.php"
    • method="get"
    • method="post"
  • <input></input>

    • A form input's most import attributes are name, value and type.

    • name: this is usually how we identify data to be passed
    • value: specifies the value of an <input> element

    • Type: Indicate what type of input we want. See a small list of input 'types' below.

      • type="text"
      • type="number"
      • type="radio"
      • type="checkbox"
      • type="submit"
