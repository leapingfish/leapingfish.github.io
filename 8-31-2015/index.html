  • box-sizing: border-box;
  • CSS Tricks Article

  • The box at the top shows the default box model. The box at the bottom shows what was once the "quirks mode" interpretation of the box model, but it now the preferred box model.

    This can be achieved using: box-sizing: border-box;

    When a specific width or height is defined this prevents the box from growing based on its padding.
