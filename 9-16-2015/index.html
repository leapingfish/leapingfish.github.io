Pseudo-Code?

  • Written in your own words, pseudo-code is the concise and simple steps that create and run your program.

Important Vocabulary

  • Statements

    • A specific instruction to be executed.
    • write ‘Hello World’

  • Variables

    • A container for storing a data values.
    • var a = 3
    • var name = "Hart"

  • Conditionals (if/else)

    • Conditionals are true/false statements used to perform different actions.
    • On the elevator, IF you press 3, go to floor three, else do nothing.
    • (greater than | less than | equal to | greater than or equal to | less than or equal to)

  • Loops (for/while)

    • Loops are used to execute the same block of code a specified number of time or while a specific condition is true.
    • while x is less than 5, execute code, increment x+1

  • Functions

    • A function is a block of code that can be called by name.
    • Functions can happen multiple times or only once.
    • Functions greatly improve the overall organization of code.
    • changeTire(this tire) = steps to change a tire
      • x = number of tires on car
      • z = 1
      • while z less than or equal to x - change this tire
